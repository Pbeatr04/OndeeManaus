# OndeeManaus

Projeto de interface web com telas de login, cadastro e escolha, desenvolvido com HTML, CSS e JavaScript.

## Tecnologias usadas
- HTML
- CSS
- JavaScript

## Arquivos do projeto
- `login.html`
- `cadastro.html`
- `escolha.html`
- `style.css`
- `script.js`

## Objetivo
Praticar desenvolvimento front-end, estrutura de páginas, estilização e interação com JavaScript.
